Project Objective

From coding exercise to professional data product

This project analyzes the City of Boston Employee Earnings Report and presents the highest earners, compensation distribution, and employees earning more than $100,000 in total earnings.

Skills Demonstrated

  • Data parsing and transformation
  • Array sorting, filtering, and aggregation
  • Dynamic HTML rendering with JavaScript
  • Responsive UI design with CSS

Methodology

How the dashboard works

1. Parse

The raw dataset is converted from indexed rows into readable employee objects.

2. Analyze

JavaScript functions calculate totals, thresholds, averages, and rankings.

3. Render

The interface updates dynamically when the user changes filters or search terms.
