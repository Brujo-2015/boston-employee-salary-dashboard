# Boston Employee Salary Dashboard

## Overview

A web-based data exploration tool that analyzes public employee salary data from the City of Boston.
The application transforms raw compensation data into structured insights, enabling quick identification of salary distribution patterns, top earners, and high-income segments.

This project demonstrates the ability to move from raw data to a clear, decision-oriented view using front-end technologies.

## Problem Statement

Public datasets are often large, unstructured, and difficult to interpret without technical tools.
This project addresses the challenge of extracting meaningful insights from raw salary data in a simple and accessible way.

---

## Solution

The application processes and visualizes employee compensation data directly in the browser, providing:

* Identification of top earners
* Count of employees above defined salary thresholds
* Average salary calculations for selected groups
* Structured ranking based on total compensation

---

## Key Features

## Architecture Approach

The project follows a simple modular front-end architecture:

* **Data Layer** → Raw JSON dataset
* **Logic Layer** → Data processing and calculations (sorting, filtering, aggregation)
* **Presentation Layer** → DOM rendering and user interface

This separation improves readability, maintainability, and scalability.
